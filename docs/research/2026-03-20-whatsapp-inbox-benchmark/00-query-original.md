# Query Original

Pesquisa de benchmark profunda sobre ferramentas de gestao de WhatsApp para equipes — foco em inbox colaborativo, organizacao de mensagens, priorizacao, resumos de grupos, anotacoes por contato e views inteligentes.

## Contexto do Projeto

**Spalla Dashboard** — plataforma de gestao para mentorias (CASE Mentoring), onde consultores gerenciam carteiras de mentorados via WhatsApp. O cenario envolve:

- Consultores com carteira de 15-30 mentorados cada
- Grupos de WhatsApp por mentorado (consultor + mentorado + equipe)
- Alto volume de mensagens em grupos que precisam ser triadas
- Necessidade de resumos de grupo para acompanhamento sem ler tudo
- Anotacoes por contato/mentorado como registro de evolucao
- Priorização inteligente: quem precisa de atenção urgente vs quem está no caminho certo
